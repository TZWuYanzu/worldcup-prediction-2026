### 冠军

> **🏆 阿根廷**

**v2 理由（Yamal 伤情是关键变量）**：

1. **阿根廷阵容唯一完整、零重大伤病**：26 人全员可用，Otamendi 38 岁仍在踢，Romero 28 岁巅峰（热刺队长 + 24-25 欧联 + MOTM），Lautaro 国米队长，Julián 马竞主力。**整个 8 强淘汰赛阶段，全队没有"带伤上阵"的核心**——这是 8 支夺冠候选里独一份
2. **马竞帮 5-6 人化学反应**：Julián / N. González / Almada / Simeone / Molina / Musso 一整赛季在 Simeone 体系下训练，**同款反抢节奏 + 同款边路反击 + 同款体能要求**——淘汰赛后段换上这帮人撕开对手是真实武器
3. **Messi / De Paul 迈阿密同俱乐部默契度**：这是别队没有的"日常合练"加成
4. **路径相对平滑**：J 头名 → R32 vs H2（乌拉圭，南美内战）→ 八强 vs L1/K2（英格兰心理脆 / 克罗地亚老化）→ 半决赛 vs K1（葡萄牙锋线薄）
5. **Scaloni 大赛胜率史诗级**：2022 WC ✅ / 2024 Copa ✅ / 2025 UNL ✅；BO1 淘汰赛几乎不犯错
6. **决赛若遇西班牙**：Yamal 带伤的话西班牙右路 1v1 武器钝化，**Dibu Martínez 在点球大战阶段是最大 buff**（2022 WC 最佳门将 + 2024 Copa 最佳门将连续两届）

### 西班牙为何不是冠军（v2 大改写）

- **Lamine Yamal 4/22 拉伤腿筋赛季报销，预计带伤参赛但状态不会在巅峰**——这是 v2 最大变量
- 失去 Yamal 右路 1v1 + 内切射门后，de la Fuente 体系"边路爆破"少一翼
- Carvajal / Morata / Joselu / Le Normand 均未入选，板凳深度受损
- **但仍是亚军大热**——Pedri + Rodri + Cubarsí + Williams 主轴足够强

### 葡萄牙为何不是冠军

- 中场最强（Vitinha 金球 3 + Bruno 21 助攻 + Bernardo + João Neves + Mendes 金球 10），但**锋线深度去 Jota 后塌陷**
- Jota 2025/07/03 西班牙车祸离世，C 罗 41 岁体能仅够禁区站位，Leão 状态不稳，Ramos/Félix 非顶级终结者
- **预测：季军**

### 法国为何不是冠军

- I 组消耗大 + Camavinga 中场缺席
- Dembélé 金球得主 + Mbappé 仍世界级，但缺乏决赛"压制对手 90 分钟"的资本
- **预测：4 强**
