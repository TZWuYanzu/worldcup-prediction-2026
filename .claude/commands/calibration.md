# 预测校准报告

运行预测评估工具，汇总整体预测表现。

## 执行步骤

### Step 1: Brier Score 报告

```bash
python3 prediction_tracker.py score
```

### Step 2: 滚动趋势

```bash
python3 prediction_tracker.py trend
```

### Step 3: 校准曲线

```bash
python3 prediction_tracker.py calibration
```

### Step 4: 预测列表

```bash
python3 prediction_tracker.py list
```

### Step 5: 汇总分析

综合以上数据输出：

1. **整体 Brier Score** — 与纯赔率基线对比，是否跑赢市场
2. **分阶段趋势** — 预测能力是否在改善（随比赛增多、方法论迭代）
3. **校准曲线** — 预测 60% 的事件是否真的发生了约 60%
4. **系统性偏差** — 是否系统性高估/低估某类结果（如高估强队、低估平局）
5. **最大偏差场次** — 哪几场预测最离谱，原因是什么

## 输出格式

```
================================================================
  预测校准报告 (截至 6月27日，N场已完赛)
================================================================

  Brier Score:
    模型: 0.XXX    赔率基线: 0.XXX    差值: +/-0.XXX
    → 模型 [优于/劣于/持平] 纯赔率

  滚动趋势:
    MD1: 0.XXX → MD2: 0.XXX → MD3: 0.XXX
    → [改善/恶化/稳定]

  校准曲线:
    预测 0-20%: 实际发生 XX% (N场)
    预测 20-40%: 实际发生 XX% (N场)
    预测 40-60%: 实际发生 XX% (N场)
    预测 60-80%: 实际发生 XX% (N场)
    预测 80-100%: 实际发生 XX% (N场)

  系统性偏差:
    - [如: 高估主队胜率 +3%]
    - [如: 低估平局概率 -5%]

  最大偏差 TOP 3:
    1. MXX: 预测 X% vs 实际 → Brier X.XXX
    2. ...
================================================================
```
