# 深度比赛复盘

对指定的已完赛比赛做密集战术拆解，超越标准 `/review` 的比分+xG汇总，进入射门级别的过程分析。

## 解析用户意图

从 `$ARGUMENTS` 中提取：
- match_id（如 `81`）→ 单场深度复盘
- 多个 match_id（如 `80 81 82`）→ 逐场深度复盘
- 球队名（如 `英格兰`）→ 该队最近一场已完赛比赛
- 无参数 → 最近一个比赛日的所有已完赛比赛

## 前置条件

比赛必须已有比分（在 `match_data.py` 中）。如果没有，先提示用户运行 `/review` 获取比分。

## 执行步骤

### Step 1: 获取逐射门 xG 时间线

```bash
python3 xg_model.py match <match_id>
```

记录每一脚射门的：时间、球队、球员、距离(m)、xG、是否进球。

### Step 2: 分阶段 xG 拆解

将射门按 15 分钟一段分组（0-15'/15-30'/30-HT/46-60'/61-75'/76-90+'，加时赛则增加 91-105'/106-120+'），计算每段的：

- 双方射门数和 xG
- 哪一方占据 xG 优势（百分比）
- 进球事件和比分走势

用 Python 脚本生成分阶段表格（参考 M81 复盘的格式）。

### Step 3: 识别战术转折点

基于分阶段数据，识别比赛中的关键转折：

**转折类型清单：**
1. **压迫→退守切换** — 某队从高xG段突然降至0xG（如M81塞内加尔62-85'零射门）
2. **闪击效应** — 低xG进球(xG<0.15)如何改变后续比赛节奏
3. **替补破局** — 换人后某队xG率显著变化
4. **蓄水决堤** — 连续多次中高xG机会未进后突然爆发（如M81比利时86'-89'）
5. **空间打开** — 对手被迫进攻后xG创造效率提升

对每个识别到的转折点，记录：
- 触发时间和原因
- 转折前后的xG率对比
- 对最终结果的因果链

### Step 4: 赛前场景推演验证

回顾 `02_data/analysis/M{id}.md` 中的赛前战术场景推演（场景A/B/C/D），逐一与实际过程对比：

- ✅ 吻合 — 过程和结果都与场景描述一致
- ◐ 部分吻合 — 结果类似但关键过程不同
- ✗ 不吻合 — 实际走势与场景描述相反
- **场景盲区** — 实际发生了但所有场景都没覆盖的关键情况

### Step 5: 关键球员微观分析

对比赛中影响力最大的2-3名球员做微观分析：
- 射门梯度（xG是否递进/递减）
- 射门位置分布（禁区内 vs 远射）
- 进球时段模式（开局型 vs 收割型）
- 与赛前分析中的预判是否一致

### Step 6: 更新分析文件

将深度复盘内容追加到 `02_data/analysis/M{id}.md` 的复盘章节，包含：
- 分阶段xG表格
- 战术转折分析
- 场景推演验证
- 关键球员微观分析

### Step 7: 更新获胜球队档案

将比赛中发现的战术特征写入/更新 memory 中的球队档案文件 `team_{team}_profile.md`：

**档案内容结构：**
- 进攻效率（场均xG + 转化率）
- 防守效率（场均xGA + 超额失球）
- 节奏特征（哪个时段最强/最弱）
- 关键球员模式（如 Kane 的"60分钟后开关"）
- 战术弱点（如面对铁桶阵的转化困难）
- 替补效应（换人后xG率变化）
- **How to apply** 段落：预测该队后续比赛时的关键变量

如果已有档案，合并新发现而非覆盖。

### Step 8: 重新生成看板

```bash
python3 generate_dashboard.py
```

## 与 /review 的区别

| 维度 | /review | /deep-review |
|------|---------|-------------|
| 范围 | 批量处理所有未复盘比赛 | 单场/少量重点比赛 |
| 深度 | 总xG + 预测对比 + Brier | 逐射门时间线 + 战术转折 |
| 耗时 | ~2分钟/场 | ~10分钟/场 |
| 输出 | 比分+方向+偏差归因 | 分阶段xG表 + 转折分析 + 球员微观 |
| 更新 | analysis文件追加简要复盘 | analysis文件追加详细复盘 + 球队档案memory |

## 关键原则

- **过程重于结果** — 不是"比分对不对"而是"比赛是怎么演进的"
- **数据说话** — 每个判断必须有对应的xG数据支撑
- **连接前后** — 与赛前分析交叉验证，发现方法论的真实盲区
- **结论谨慎性** — 单场样本不足以得出系统性结论，≥5场样本才提升为方法论教训，3场以下标注为"待观察信号"
- **档案积累** — 每次深度复盘都是在完善球队画像，为后续预测提供更精准的输入
