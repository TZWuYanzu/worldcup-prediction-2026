# 淘汰赛晋级推演

基于本届世界杯表现（xG + Elo），通过蒙特卡洛模拟（10,000次）计算R32→R16→QF→SF→Final各轮晋级概率和夺冠概率。

## 解析用户意图

从 `$ARGUMENTS` 中提取：
- 无参数 → 运行完整推演（全部32支球队）
- 球队名（如 `巴西`、`Brazil`、`阿根廷`）→ 该队详细晋级路径
- `update` → 更新已完赛结果后重新推演

## 执行步骤

### Step 1: 运行蒙特卡洛模拟

**完整推演：**
```bash
python3 knockout_simulator.py
```

**单队路径：**
```bash
python3 knockout_simulator.py --team <球队名>
```

### Step 2: 结合本届表现给出定性分析

根据模拟结果，补充以下分析：

1. **夺冠热门**（TOP 5）— 结合小组赛xG表现、Elo变化、阵容深度，解释为何概率高
2. **黑马预警** — 概率虽低但有爆冷潜力的球队（如摩洛哥2022先例）
3. **半区分析** — 上下半区实力是否均衡，哪个半区更拥挤
4. **关键对决** — R16/QF阶段最具看点的潜在对阵

### 关键原则

- **数据驱动** — 概率基于Elo和模型预测，不因名气偏移
- **已完赛比赛** — 直接取实际结果，不再模拟
- **淘汰赛校准** — D概率下限25%，反映淘汰赛实际平局率
- **模型预测优先** — 有predictions.json中的预测时优先使用，无预测时用Elo估算
